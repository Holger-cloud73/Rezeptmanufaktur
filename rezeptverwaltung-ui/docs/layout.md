# Layout

## Desktop

Die Anwendung wird als zentrierter App-Container dargestellt.

Maximale Breite pro Screen:
- Homepage: 390px bis 420px
- Formularseite: 420px bis 460px
- Detailseite: 420px bis 460px

In der Präsentationsansicht können drei Screens nebeneinander stehen.

## Mobile

- Eine Seite pro Bildschirm
- Volle Breite
- Padding 16px
- Cards stapeln sich vertikal

## Breakpoints

- Mobile: bis 640px
- Tablet: 641px bis 1024px
- Desktop: ab 1025px

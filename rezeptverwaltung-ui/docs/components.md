# Components

## Header

- Weißer Hintergrund
- Links Logo mit Kochmütze-Icon
- Rechts Burger-Menü
- Höhe ca. 56px
- Leichter Schatten oder Border unten

## Hero Slider

- Großes Rezeptbild
- Dunkler Verlauf über dem Bild
- Weißer Text links
- Runde Slider-Punkte unten
- Pfeile links und rechts

## Category Card

- Bild oben
- Dunkler Overlay-Verlauf
- Kategorie-Name unten
- Abgerundete Ecken
- Hover: leichtes Vergrößern

## Recipe Card

- Bild
- Rezeptname
- kleine Beschriftung
- abgerundete Ecken
- Schatten

## Form Input

- Hellgrauer Rahmen
- Radius 6px
- Höhe ca. 36px
- Fokus: grüner Rahmen

## Button Primary

- Grüner Hintergrund
- Weißer Text
- Radius 6px
- Hover dunkler

## Recipe Detail Layout

- Großes Bild oben
- Darunter zweispaltiges Layout:
  - Links Beschreibung, Zutaten, Zubereitung
  - Rechts Zeiten und Schwierigkeit

# Claude Instructions

Du arbeitest an einer statischen HTML/CSS-Vorlage für eine Rezeptverwaltungs-Webanwendung.

## Ziel

Baue das Webdesign aus der Referenzdatei:

references/rezeptverwaltung-vorlage.png

Die Anwendung besteht aus drei Seiten:

1. Homepage
2. Rezept erstellen
3. Rezept anzeigen

## Wichtige Dokumente

Lies zuerst:

- docs/ui-spec.md
- docs/design-tokens.md
- docs/components.md
- docs/layout.md
- docs/assets.md
- docs/tasks.md

## Technische Regeln

- Verwende nur HTML, CSS und optional wenig Vanilla JavaScript.
- Kein React, kein Angular, kein Next.js.
- Keine externen CSS-Frameworks.
- Kein Bootstrap.
- Schreibe sauberes, gut kommentiertes CSS.
- Nutze CSS Custom Properties.
- Mobile-first, aber Desktop soll gut aussehen.
- Erstelle realistische Platzhalterbilder, falls keine finalen Bilder vorhanden sind.
- Achte auf semantisches HTML.
